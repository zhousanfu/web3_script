# config (你需要的配置)
复制.env.example 重新命名为.env

填入各个项目的参数
```
CRASS_USERID = ''

GAEA_UID = ''
GAEA_TOKEN = ''

NODEPAY_TOKEN = ''

DAWN_USER = ''
DAWN_PASS = ''
```

代理保存在: Proxies/proxies.txt
